Web Designer Taro Ogawa

東京都在住のWebデザイナー，小川太郎です。オンラインスクールでWeb制作について学び，出版業界からWeb業界へと転職活動中です。これからさらにプログラミングの技術や知識を身につけて，データベース構築を含めたサイト運営ができるWebエンジニアになりたいと思っています。

Skills

Web

HTML/CSSでのコーディングができます。JavaScript，PHP，WordPressも現在勉強中です。

Writing

文章を書くことが得意です。Webサイトをコンテンツからデザインまでトータルにコーディネートいたします。

Design

DTPデザインの経験があります。チラシ，フライヤー，ポスターなどの制作も承ります。
